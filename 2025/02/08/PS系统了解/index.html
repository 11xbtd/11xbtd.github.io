发表于2025-02-08|更新于2025-02-08|PeopleSoft
|总字数:939|阅读时长:5分钟|浏览量:

PS系统一些名称概念

岗位编号—-职务编号 (position_nbr)

标准岗位编号 —- 职位编号(job_code)

empl_rcd – 岗位标识(是否兼岗,0兼岗)

hr_status – 是否在职(A在职,I离职)


PS系统的主键都是定义在AD工具中,不在数据库中体现

系统变量

变量名 解释
%SignonUserId returns the value the user typed in at the sign on page.
%OperatorId 当前用户id
%UserDescription returns the description (if any) listed for the current user.
%UserId returns a character string containing the user currently logged on.
%This returns a reference to the current object.
%Session returns a reference to the current, existing session. If you use %Session successfully, you do not have to use the GetSession function and Connect method. If you do not have a current session, %Session returns NULL.
%Response returns a reference to the response object. This reference can be used like an object, that is, you can use this as part of a dot notation string. For example: &CookieArray = %Response.CookieNames();
%Response 同上
%Time 检索 当前数据库服务器时间。
%IntBroker 使用 %IntBroker 系统 变量返回对 Web 服务网关对象的引用。
%Date 返回一个 Date 值等于当前服务器日期。这可能是不同的 比 %ClientDate 返回的客户端日期。您可以使用此系统 variable 作为日期字段的默认值。
%DateTime 返回 当前服务器日期和时间作为 Datetime 值。
